# Шрифт Phoenicia Condensed Italic

Для использования шрифта Phoenicia Condensed Italic в хэдере и футере сайта необходимо скачать файлы шрифта и поместить их в эту папку.

## Инструкция по установке:

1. Скачайте файлы шрифта Phoenicia Condensed Italic с одного из источников:
   - https://ffont.ru/font/phoeniciacondenseditalic
   - https://ru.fonts2u.com/phoenicia-condensed-italic.шрифт
   - Другие источники с бесплатными шрифтами

2. Поместите файлы шрифта в эту папку (`static/fonts/`) со следующими именами:
   - `PhoeniciaCondensed-Italic.woff2` (предпочтительно)
   - `PhoeniciaCondensed-Italic.woff`
   - `PhoeniciaCondensed-Italic.ttf`
   - `PhoeniciaCondensed-Italic.otf`

3. После добавления файлов шрифт автоматически начнет использоваться в хэдере и футере сайта.

## Примечание:

Если файлы шрифта не найдены, браузер будет использовать fallback шрифты:
- Rajdhani (из Google Fonts)
- Orbitron (из Google Fonts)
- Arial Narrow (системный шрифт)

Эти шрифты похожи по стилю, но не идентичны Phoenicia Condensed.
