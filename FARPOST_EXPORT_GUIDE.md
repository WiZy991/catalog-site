# Руководство по выгрузке прайсов на Farpost

## ⚠️ Важно: Двухэтапный процесс

Согласно ответу от Farpost, процесс состоит из двух этапов:

1. **Первая выгрузка (вручную)** - разместить прайс на Farpost вручную
2. **Последующие синхронизации (автоматически)** - синхронизировать через API

---

## 📋 Этап 1: Первая выгрузка прайса на Farpost

**Farpost подключает API к уже размещенному прайсу.** Это значит:

### Шаг 1: Экспорт прайса с сайта

1. Зайдите в админ-панель → раздел **"Товары"**
2. Выберите товары для выгрузки:
   - Можно выбрать все товары (галочка вверху списка)
   - Можно выбрать конкретные товары (галочки слева)
   - Можно использовать фильтры для выбора по категории, бренду и т.д.
3. В выпадающем списке **"Действие"** выберите **"Экспорт для Farpost"**
4. Скачайте CSV файл с товарами

### Шаг 2: Размещение прайса на Farpost

1. Зайдите на сайт Farpost в раздел личного кабинета
2. Создайте **пакет-объявление** (или используйте существующий)
3. Загрузите скачанный CSV файл через интерфейс Farpost
4. Разместите прайс на площадке

### Шаг 3: Подключение API от Farpost

1. Свяжитесь с Farpost и попросите подключить API к вашему размещенному прайсу
2. Farpost подключит API со своей стороны
3. После подключения Farpost предоставит вам данные для API:
   - **Логин** для API
   - **Пароль** для API  
   - **ID пакет-объявления** (packet_id)

### Шаг 4: Как найти ID пакет-объявления (packet_id)

После размещения прайса, ID пакет-объявления можно найти в URL:

```
https://www.farpost.ru/personal/goods/packet/{ID}/recurrent-update
```

Например, если URL выглядит так:
```
https://www.farpost.ru/personal/goods/packet/123456/recurrent-update
```

То `packet_id = 123456`

> 💡 **Важно:** Один пакет-объявление может содержать множество товаров (тысячи) из разных категорий. Farpost подключает API именно к этому пакету.

---

## 📋 Этап 2: Настройка автоматического обновления

После того, как Farpost подключит API, у вас есть два способа обновления:

### Способ 1: Автоматическое обновление через URL (рекомендуется) ⭐

**Farpost сам периодически загружает прайс-лист по указанной ссылке!**

1. **На вашем сайте доступны публичные URL для прайс-листа:**
   - CSV: `https://ваш-сайт.ru/farpost/price-list.csv`
   - XLS: `https://ваш-сайт.ru/farpost/price-list.xls`
   - XML: `https://ваш-сайт.ru/farpost/price-list.xml`

2. **В личном кабинете Farpost:**
   - Перейдите на страницу вашего прайс-листа
   - Откройте вкладку **"Автоматически"**
   - Выберите **"Периодически обновлять прайс-лист по ссылке ниже"**
   - Вставьте ссылку на ваш прайс-лист (например: `https://ваш-сайт.ru/farpost/price-list.csv`)
   - Нажмите **"Сохранить"**

3. **Farpost автоматически будет загружать прайс-лист** по указанной ссылке с заданной периодичностью.

> 💡 **Преимущества:** Не нужно настраивать cron, не нужно управлять обновлениями - Farpost делает всё сам!

### Способ 2: Ручная синхронизация через API

Для ручной синхронизации через API настройте учетные данные в админ-панели:

### Шаг 1: Зайдите в админ-панель сайта

Войдите в админ-панель Django с правами администратора.

### Шаг 2: Настройте API Farpost

1. В меню админ-панели найдите раздел **"Настройки API Farpost"** (или **"FarpostAPISettings"**)
2. Нажмите **"Добавить настройки API Farpost"**
3. Заполните форму:
   - **Ключ API** (предпочтительно) - ключ, предоставленный Farpost для API. Используется для расчета auth (SHA512 от ключа). Если указан, имеет приоритет над login:password.
   - **Логин** и **Пароль** (альтернатива) - если ключ API не предоставлен, можно использовать логин и пароль (пароль будет сохранен в зашифрованном виде)
   - **ID пакет-объявления** - ID вашего уже размещенного прайса на Farpost (см. этап 1)
   - **Автоматическое обновление** - (опционально) для использования management command
   - **Ссылка на прайс-лист** - (опционально) для management command
   - **Интервал обновления (часы)** - (опционально) для management command
   
   > **Примечание:** Для автоматического обновления через Farpost используйте публичный URL:
   > `https://ваш-сайт.ru/farpost/price-list.csv` (или .xls, .xml)
   > Укажите этот URL в настройках Farpost в личном кабинете.
   - **Активен** - отметьте галочку, чтобы использовать эти настройки
4. Нажмите **"Сохранить"**

> 💡 **Совет:** Можно создать несколько настроек с разными `packet_id`, если нужно синхронизировать разные пакеты.

## 🚀 Ручная синхронизация через API

После того как Farpost подключил API, вы можете синхронизировать изменения вручную через админ-панель:

### Инструкция по синхронизации:

1. Зайдите в раздел **"Товары"** в админ-панели
2. Выберите товары для синхронизации:
   - Можно выбрать все товары (галочка вверху списка)
   - Можно выбрать конкретные товары (галочки слева от каждого товара)
   - Можно использовать фильтры для выбора товаров по категории, бренду и т.д.
3. В выпадающем списке **"Действие"** выберите **"Синхронизировать с API Farpost"**
4. Нажмите кнопку **"Выполнить"**
5. Система автоматически:
   - Сформирует файл с данными товаров (в формате XLS)
   - Отправит файл в API Farpost
   - Обновит данные на Farpost
   - Покажет результат синхронизации

### Ограничения:

- API Farpost принимает файлы размером **до 5 МБ**
- Примерно **~3000-4000 товаров** помещается в один файл
- Если товаров больше, разбейте синхронизацию на партии:
  - Используйте фильтры для выбора товаров партиями
  - Например, синхронизируйте по категориям или по брендам

### Формат выгружаемых данных:

Каждый товар синхронизируется с:
- **Заголовок** - автоматически сгенерированный (Бренд + Артикул + Характеристика)
- **Цена** - текущая цена товара
- **Описание** - полное описание с характеристиками, применимостью, кросс-номерами
- **Артикул** и **Бренд**
- **Фотографии** - до 5 изображений (ссылки на изображения с вашего сайта)
- **Ссылка на сайт** - прямая ссылка на карточку товара на вашем сайте
- **Состояние** и **Наличие**

> ⚠️ **Важно:** API Farpost обновляет только **уже существующие** объявления в пакете. Для добавления новых товаров может потребоваться повторная ручная загрузка CSV или создание нового пакета.

## 📊 Мониторинг синхронизации

После синхронизации система показывает:
- ✅ **Успешно** - товары отправлены на Farpost
- ❌ **Ошибка** - что-то пошло не так (проверьте сообщение об ошибке)

### Статус последней синхронизации

В настройках API Farpost можно посмотреть:
- **Последняя синхронизация** - когда была последняя выгрузка
- **Статус последней синхронизации** - успешно или ошибка
- **Ошибка последней синхронизации** - детали ошибки (если была)

## 🔧 Технические детали

### API Endpoint Farpost

```
POST https://www.farpost.ru/good/packet/api/sync
```

### Аутентификация

Система поддерживает два способа аутентификации:
1. **Ключ API** (предпочтительно) - SHA512 хеш от ключа, предоставленного Farpost
2. **Логин:Пароль** (для обратной совместимости) - SHA512 хеш от строки `логин:пароль`

Согласно инструкции Farpost, ключ API предоставляется по запросу и является предпочтительным способом аутентификации.

### Поддерживаемые форматы

- **XLS** (по умолчанию) - самый компактный формат
- **CSV** - текстовый формат
- **XML** - структурированный формат

### Процесс синхронизации

1. Система формирует файл с выбранными товарами
2. Генерирует абсолютные URL для изображений товаров
3. Создает хеш аутентификации (SHA512 от ключа API или логин:пароль)
4. Отправляет POST запрос на API Farpost с файлом
5. Получает ответ и сохраняет статус синхронизации

### Автоматическое обновление

**Важно:** Farpost сам периодически загружает прайс-лист по указанной ссылке!

Для настройки автоматического обновления:

1. **Создайте публичный URL для прайс-листа на вашем сайте:**
   - CSV: `https://ваш-сайт.ru/farpost/price-list.csv`
   - XLS: `https://ваш-сайт.ru/farpost/price-list.xls`
   - XML: `https://ваш-сайт.ru/farpost/price-list.xml`

2. **В личном кабинете Farpost:**
   - Перейдите на страницу прайс-листа
   - Откройте вкладку **"Автоматически"**
   - Выберите **"Периодически обновлять прайс-лист по ссылке ниже"**
   - Вставьте ссылку на ваш прайс-лист (например: `https://ваш-сайт.ru/farpost/price-list.csv`)
   - Нажмите **"Сохранить"**

3. **Farpost автоматически будет загружать прайс-лист** по указанной ссылке с заданной периодичностью.

> 💡 **Примечание:** Management command `update_farpost_price_list` можно использовать для принудительного обновления через API, но основной способ - это автоматическая загрузка Farpost по ссылке.

## ❓ Часто задаваемые вопросы

### Как часто можно синхронизировать?

Синхронизировать можно в любое время и с любой частотой. Система автоматически обновляет данные на Farpost.

### Что если товаров больше 4000?

Разбейте выгрузку на партии:
1. Используйте фильтры в админ-панели (по категории, бренду и т.д.)
2. Выгружайте по 3000-4000 товаров за раз
3. Повторяйте для следующих партий

### Можно ли автоматизировать синхронизацию?

Да, можно создать Django management команду и настроить cron для автоматической синхронизации. Для этого обратитесь к разработчику.

### Что делать, если синхронизация не работает?

1. Проверьте настройки API Farpost:
   - Логин и пароль правильные?
   - Packet ID указан верно?
   - Настройки активны (галочка "Активен")?

2. Проверьте размер файла:
   - Если товаров слишком много, разделите на партии

3. Посмотрите детали ошибки:
   - В админ-панели → Настройки API Farpost → Поле "Ошибка последней синхронизации"

## 🎯 Итого

### Можно ли выгрузить прайс на площадку?

**Да, но в два этапа:**

#### Этап 1: Первая выгрузка (вручную) ✅
1. Экспортируйте прайс с сайта: Товары → "Экспорт для Farpost" → Скачайте CSV
2. Разместите CSV на Farpost через их интерфейс
3. Попросите Farpost подключить API к вашему прайсу

#### Этап 2: Последующие синхронизации (автоматически) ✅
1. Получите от Farpost: логин, пароль и packet_id
2. Настройте API в админ-панели: "Настройки API Farpost"
3. Синхронизируйте изменения: Товары → "Синхронизировать с API Farpost"

### Время выгрузки:

**Ручной экспорт (CSV):**
- ~1000 товаров: мгновенно (скачивание файла)
- Загрузка на Farpost зависит от их интерфейса

**Автоматическая синхронизация (API):**
- ~1000 товаров: 1-2 минуты
- ~3000 товаров: 3-5 минут
- Зависит от размера описаний и количества изображений

### Резюме процесса:

1. **Первая выгрузка** → Экспорт CSV → Ручная загрузка на Farpost → Подключение API
2. **Обновления** → Автоматическая синхронизация через API

> 💡 **Совет:** После первой выгрузки все изменения (цены, наличие, описания) можно синхронизировать автоматически через API! 🚀
