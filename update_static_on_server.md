# Инструкция по обновлению статических файлов на продакшене

После изменений в CSS файлах нужно обновить статические файлы на сервере.

## Шаги для обновления:

```bash
# 1. Перейдите в директорию проекта
cd /home/o/onesim8n/onesimus/onesimus

# 2. Активируйте виртуальное окружение
source venv/bin/activate

# 3. Соберите статические файлы
python manage.py collectstatic --noinput

# 4. Если используете nginx/apache, перезапустите сервер (если нужно)
# или просто очистите кеш браузера на продакшене

# 5. Для проверки очистите кеш браузера или откройте в режиме инкогнито
```

## Что было изменено:

1. **Иконка корзины** - добавлены стили для квадратной формы (44x44px с !important)
2. **Изображения категорий** - добавлен `border-radius: 50%` для круглых изображений

## Если изменения не применяются:

1. Очистите кеш браузера (Ctrl+Shift+Delete или Cmd+Shift+Delete)
2. Откройте сайт в режиме инкогнито
3. Проверьте, что файлы действительно обновились на сервере
4. Убедитесь, что nginx/apache обслуживает правильную директорию со статикой
