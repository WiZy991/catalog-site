# Формат импорта товаров

## Поддерживаемый формат файла клиента

Система настроена для работы с Excel файлами в следующем формате:

### Структура файла:

1. **Заголовки** могут находиться в любой из первых 10 строк (система автоматически найдет строку с заголовками)
2. **Колонки:**
   - **Артикул** - может быть пустой, тогда артикул извлекается из названия
   - **Номенклатура, Характеристика. Наименование для печати** - полное название товара
   - **Розничная Фарпост RUB Не включает Цена** - цена товара
   - **Склад Уссурийск Остаток** - остаток на складе

### Формат названия товара:

```
Амортизатор, DAIHATSU 332120 /48510-B1020 M300/M301 F/R/L 2WD
```

Структура:
- **Тип товара**: Амортизатор (определяется категория)
- **Бренд**: DAIHATSU (извлекается автоматически)
- **Артикул**: 332120 (6-значное число после бренда)
- **OEM номер**: 48510-B1020 (после слеша, сохраняется в характеристики)
- **Применимость**: M300/M301 F/R/L 2WD (все после OEM номера)

### Формат цены:

- `2 000,00` - пробел как разделитель тысяч, запятая как десятичный разделитель
- Результат: 2000.00 рублей

### Формат количества:

- `4,000` - запятая как разделитель тысяч (целое число)
- Результат: 4000 единиц

### Автоматическое определение:

- **Категория**: по ключевым словам (Амортизатор → Подвеска)
- **Бренд**: из известных брендов (DAIHATSU, HONDA, INFINITI, ISUZU, KIA, MAZDA и др.)
- **Артикул**: 6-значное число после бренда
- **OEM номер**: код после слеша
- **Применимость**: все после OEM номера

### Наличие товара:

Автоматически определяется по остатку:
- Остаток > 0 → "В наличии"
- Остаток = 0 → "Нет в наличии"
