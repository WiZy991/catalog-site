# СРОЧНО: Применить миграции на сервере

## Команды для выполнения на сервере:

```bash
cd /home/o/onesim8n/onesimus/onesimus
source venv/bin/activate
git pull origin main
python manage.py migrate catalog
touch tmp/restart.txt
```

## Что это исправит:

- Создаст таблицу `catalog_productcharacteristic` (исправит ошибку при удалении товаров)
- Создаст таблицу `catalog_synclog` (логи синхронизации)
- Добавит поля для автоматического обновления Farpost

## Проверка после применения:

```bash
python manage.py showmigrations catalog
```

Все миграции должны быть отмечены как `[X]` (применены).
