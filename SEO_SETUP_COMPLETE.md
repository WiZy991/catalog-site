## 🔧 Что нужно сделать вручную:

### 1. **Яндекс.Вебмастер** (https://webmaster.yandex.ru/)

1. Зарегистрируйтесь в Яндекс.Вебмастер
2. Добавьте сайт:
   - `onesimus25.ru`
   - `www.onesimus25.ru` (если нужно)
3. Выберите способ верификации "HTML-тег"
4. Скопируйте код верификации (например: `abc123def456`)
5. Вставьте код в `config/settings.py`:
   ```python
   YANDEX_VERIFICATION = 'ваш_код_верификации'
   ```
6. Отправьте sitemap: `https://onesimus25.ru/sitemap.xml`

### 2. **Google Search Console** (https://search.google.com/search-console)

1. Зарегистрируйтесь в Google Search Console
2. Добавьте сайт:
   - `onesimus25.ru`
   - `www.onesimus25.ru` (если нужно)
3. Выберите способ верификации "HTML-тег"
4. Скопируйте код верификации (например: `abc123def4567890`)
5. Вставьте код в `config/settings.py`:
   ```python
   GOOGLE_VERIFICATION = 'ваш_код_верификации'
   ```
6. Отправьте sitemap: `https://onesimus25.ru/sitemap.xml`

### 3. **Яндекс.Метрика** (опционально, но рекомендуется)

1. Создайте счетчик в Яндекс.Метрике (https://metrika.yandex.ru/)
2. Скопируйте ID счетчика (число, например: `12345678`)
3. Вставьте ID в `config/settings.py`:
   ```python
   YANDEX_METRICA_ID = '12345678'
   ```

## 🎯 Особенности реализации:

### Автоматическое переключение доменов
Система автоматически определяет, какой домен использовать как канонический:
- Если запрос идет с `onesimus25.ru` → используется `onesimus25.ru`
- Если запрос идет с `www.onesimus25.ru` → используется `www.onesimus25.ru`
- Если запрос идет с временного домена → используется временный домен

### Canonical URLs
Все страницы имеют правильные canonical URLs, которые:
- Указывают на канонический домен
- Предотвращают проблемы с дублями контента
- Помогают поисковым системам понять, какая версия страницы основная

### Schema.org разметка
Полная структурированная разметка для:
- Лучшего понимания контента поисковыми системами
- Отображения расширенных сниппетов в поиске
- Улучшения CTR в результатах поиска

## ✅ Готово к использованию!

После добавления кодов верификации в `config/settings.py` SEO будет полностью настроено и готово к работе!
