# Инструкция по настройке и использованию

## 1. Выполните миграции

```bash
python manage.py makemigrations
python manage.py migrate
```

## 2. Настройка email для заказов

В `config/settings.py` уже настроен `EMAIL_BACKEND = 'django.core.mail.backends.console.EmailBackend'` для разработки.

**Для продакшена** раскомментируйте и настройте SMTP:

```python
EMAIL_BACKEND = 'django.core.mail.backends.smtp.EmailBackend'
EMAIL_HOST = 'smtp.mail.ru'
EMAIL_PORT = 587
EMAIL_USE_TLS = True
EMAIL_HOST_USER = 'onesimus25@mail.ru'
EMAIL_HOST_PASSWORD = 'your-password-here'
```

## 3. Использование корзины

- Товары добавляются в корзину через кнопку "В корзину" на карточках товаров
- Корзина хранится в сессии пользователя
- При оформлении заказа данные отправляются на email менеджера (`onesimus25@mail.ru`)

## 4. Экспорт для Farpost

1. В админ-панели перейдите в раздел "Товары"
2. Выберите товары для экспорта
3. В выпадающем списке "Действие" выберите "Экспорт для Farpost"
4. Скачайте CSV файл

Экспорт включает:
- Автоматически сгенерированные заголовки (Бренд + Артикул + Характеристика)
- Структурированные описания
- Ссылки на изображения (до 5 фото)
- Уникальные ссылки на карточки товара на сайте
- Все характеристики, применимость, кросс-номера

## 5. Карта на странице контактов

Используется Яндекс.Карты. Для работы нужен доступ к интернету.

Координаты настроены для: **Приморский край, г. Уссурийск, ул. Котовского 17В**

## 6. Название компании

Название компании **Onesimus** установлено в настройках и отображается:
- В шапке сайта
- На всех страницах
- В админ-панели

## 7. Добавление логотипа компании

1. Поместите файл логотипа в папку `static/images/` с именем `logo.png`
   - Рекомендуемые форматы: PNG (с прозрачным фоном) или SVG
   - Рекомендуемые размеры: ширина 200-300px, высота до 100px

2. Логотип автоматически появится:
   - В шапке сайта (вместо эмодзи)
   - В футере сайта
   - В админ-панели Django Jazzmin

3. Если логотип не нужен, закомментируйте или удалите строку `SITE_LOGO = 'images/logo.png'` в `config/settings.py`
