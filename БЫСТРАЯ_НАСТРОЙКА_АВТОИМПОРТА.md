# 🚀 Быстрая настройка автоматического импорта

## ⚡ За 3 шага

### Шаг 1: Загрузите обновленные файлы на сервер

```bash
cd ~/onesimus/onesimus
git pull
# Или загрузите вручную:
# - catalog/management/commands/process_1c_files.py
```

### Шаг 2: Настройте cron

```bash
# Откройте crontab
crontab -e

# Добавьте эту строку (проверка каждые 5 минут):
*/5 * * * * cd /home/o/onesim8n/onesimus/onesimus && /home/o/onesim8n/onesimus/venv/bin/python manage.py process_1c_files --recent 10 >> /home/o/onesim8n/onesimus/onesimus/logs/cron_1c_import.log 2>&1

# Сохраните (Ctrl+O, Enter, Ctrl+X в nano)
```

**Важно:** Замените пути на ваши реальные пути!

### Шаг 3: Проверьте работу

```bash
# Проверьте, что задача добавлена
crontab -l

# Проверьте логи
tail -f logs/cron_1c_import.log
```

## ✅ Готово!

Теперь после нажатия "Выполнить обмен" в 1С:
1. Файл сохраняется в `media/1c_exchange/`
2. Через 5 минут (или меньше) cron находит файл
3. Автоматически обрабатывает и обновляет товары
4. Вам больше не нужно ничего делать вручную! 🎉
